# Modular Web Core

Modular Web Core is a lightweight PHP-based engine for building modular, template-driven web systems.

## Philosophy

- Clean architecture
- File-based modular structure
- Lightweight deployment
- No framework dependency

## Features

- Modular content blocks
- Clean routing system
- Admin interface
- Template separation
- Deployment-ready structure

## Project Structure (planned)

/core
/modules
/templates
/public
/config
